Metody w obu wersjach (C#, Python):
- tworzenie playlist (nazwa playlisty nie może przekraczać 16 znaków)
- usuwanie playlisty poprzez wpisanie tytułu playlisty, którą chcemy usunąć
- dodanie tracku do playlisty
- usunięcie tracku z playlisty, poprzez wpisanie tytułu tracku, który chcemy usunąć
- odtwarzanie playlisty albo w normalnym trybie (po kolei) albo w trybie shuffle (tracki są ze sobą mieszane, powstaje tymczasowa nowa lista tracków z nową kolejnością)
- metoda showPlaylist wypisuje wszystkie dostępne playlisty (ich tytuły, ilość tracków oraz całkowity czas trwania całej playlisty (inspiracja od Spotify),
a następnie daje listę tracków z podstawowymi informacjami o każdym z nich)

W dodatku jest obsługa błędów z ich numeracją, właściwości tracków (tytuł, wykonawca, czas odtwarzania, gatunek, data powstania).
